1 3ds max

1.1 基本操作

快捷键 功能
按下鼠标滚轮拖动 拖动视图
Alt+按下鼠标滚轮拖动 旋转视图
Alt+w 最大化、最小化视图切换
g 显示、隐藏场景中网格线
h 按名称选择物体
l t f u p 左-顶-前-正交-透视视图切换
F3 线框模式显示切换
F4 EdgedFaces 模式显示切换

1.2 动画

1.2.1 基础知识

关键帧类型 颜色 功能
Planted Key (踩踏/固定关键帧) 橙色 站军姿 等动作，为保持一动不动时使用
Sliding Key(滑动关键点) 黄色 走路、跑步等运动中任何一直接触地面的部位都用滑动关键帧
Free Key(自由关键帧) 黑色 运动中不会一直接触地面的部位都用自由关键帧

1.2.2 骨骼绑定步骤

  1. 分析模型

    性别 女
    个性 豪放
    职业 坦克(肉盾) 保护队友 控制
    形体 人形 两足(无残缺)

  2. 分析模型需要活动的部位和道具
    1. CS 人形骨骼自带

      手
      脚
      头
      躯干

    2. Bones 骨骼

      盾牌
      武器
      头发
      飘带(衣服的裙摆)

  3. 骨骼配置
    1. 确定骨骼重心 与模型重心吻合
    2. 配置腰或者腿骨骼 让骨骼关节和模型关节吻合
    3. 根据模型布线来确定躯干
    4. 确定肩 骨骼肩膀末端（靠近胸的一端）的旋转点要在模型锁骨位置，前端与模型大臂相吻合
    5. 配置手臂，手，手指
    6. 配置头

1.3.3 Tips

  1. print 比较诡异
    print "hello ->" + "world"
-- output: 输出两次
-- "hello ->"
-- "hello ->world"

print("hello ->" + "world")
-- output: 输出两次
-- "hello ->world"
-- "hello ->world"
  2. maxscript 变量不分大小写
    -- 下面会重名
local tmpIdx = 1
local TmpIdx = 2
  3. 删除老的 macroscript

    删除#userScripts 目录下的脚本即可。
    #userScripts 的值为 "C:\Users\<UserName>\AppData\Local\Autodesk\3dsMax\<ReleaseNumber> - 64bit\ENU\scripts"
    https://knowledge.autodesk.com/search-result/caas/CloudHelp/cloudhelp/2015/ENU/MAXScript-Help/files/GUID-F7577416-051E-478C-BB5D-81243BAAC8EC-htm.html

  4. maxscript –Type error: Call needs function or class, got:undefined

    安装 maxscript 后，重启 3ds max，执行安装的脚本，报错：maxscript –Type error: Call needs function or class, got:undefined
    解决方案：
    将脚本放到 C:\Program Files\Autodesk\3ds Max 2014\stdplugs\stdscripts 目录下安装就可以了

  5. – No ""eof"" function for undefined <<

    打开文件时报如上错误。
    解决方案：
    查看被打开的文件是否被其他程序打开了，将其他程序关闭再执行。

  6. Runtime error: Cannot directly set animation context time - use "at time…".
    -- 不能直接设置 currentTime
currentTime = 10
-- 需要使用 sliderTime
sliderTime = 10
