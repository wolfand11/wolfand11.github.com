打印栈相关的信息

bt                   # backtrace 打印整个调用栈的回溯。它会显示每个栈帧的索引、函数名、源代码文件以及行号（如果可用）
f 0                  # 查看栈中特定帧的信息，可以使用这个命令选择一个不同的栈帧。可以通过帧索引或地址来选择
reg read pc          # 查看 PC（程序计数器）
reg read rbp         # 查看基址指针
reg read rsp         # 查看栈顶指针
x/4wx $rbp           # 显示rbp 指向的地址开始的4个word， 以16进制显示内容
x/8wx `$rbp-4*8`     # 从rbp 往低地址方向走4个word（1个word 8个字节）到达目的地址，显示目的地址开始的8个word， 以16进制显示内容

p myVariable         # 显示局部变量 myVariable
